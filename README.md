# TSP solver
Travelling salesman problem solver written in Rust using [tsplib](https://github.com/mastqe/tsplib).

## Quick Start

```bash
git clone https://github.com/jaroslavszkandera/tsp-solver.git
cd tsp-solver
git clone https://github.com/mastqe/tsplib
cargo run
```
